# wonders-demo

> Demo application made by [Wonders](https://github.com/vutran/wonders).

## Install

```bash
$ npm i -g wonders-demo
```

## Usage

```bash
# Prints "Beep!"

$ wonders beep

# Prints "Boop!"

$ wonders boop

# Search for a repository

$ wonders search <repoName>
```
